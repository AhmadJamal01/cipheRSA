## Getting Started

### Prerequisites

Install Flask
```sh
   python -m pip install flask
   ```
Install Flask-SocketIO
```sh
   python -m pip install flask-socketio
   ```
Install Eventlet 
  ```sh
   python -m pip install eventlet
   ```
Clone the repo and run app.py  ..  Now you have setup the backend, it is time for Chatting ðŸ‘½
